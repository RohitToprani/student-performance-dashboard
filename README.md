Student Performance Dashboard

Project Overview:
This project is an interactive Excel dashboard built to analyze student performance across subjects.

Tools Used:
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- KPI Metrics

Key Features:
- Average Marks by Subject
- Pass vs Fail Analysis
- Performance Level Breakdown (Top, Average, Weak)
- Interactive filtering using slicers

Key Insights:
- 80% of students passed the exam
- Maths and Chemistry show strong performance
- Weak students are concentrated in specific subjects

Purpose:
This dashboard helps in identifying student performance trends and areas needing improvement.
